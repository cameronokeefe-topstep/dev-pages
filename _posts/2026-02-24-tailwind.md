## Tailwind CSS Cheatsheet

A practical reference for common **Tailwind CSS** utility classes (Tailwind v3+).

---

## 1. Layout

### Container

```html
<div class="container mx-auto px-4"></div>
```

### Display

```md
block | inline-block | inline | flex | inline-flex | grid | hidden
```

```html
<div class="flex"></div>
<div class="grid"></div>
```

### Position

```md
static | relative | absolute | fixed | sticky
```

```html
<div class="relative">
  <div class="absolute top-0 right-0"></div>
</div>
```

### Z-Index

```html
<div class="z-10"></div>
```

## 2. Flexbox

### Direction & Wrap

## 16. State Variants

```md

```

## 18. Common UI Patterns

### Card

```html
<div class="rounded-2xl border border-gray-200 bg-white p-6 shadow-md">
  <h3 class="text-lg font-semibold">Title</h3>
  <p class="mt-2 text-sm text-gray-600">Body text</p>
</div>
```

### Button

```html
<button class="inline-flex items-center justify-center rounded-md bg-blue-600 px-4 py-2 text-white hover:bg-blue-700 active:scale-95 transition">
  Click
</button>
```

### Input + Label

```html
<label class="block text-sm font-medium text-gray-700">Email</label>
<input class="mt-1 w-full rounded-md border border-gray-300 px-3 py-2" />
```

## 20. Gotchas & Tips

* Prefer `gap-*` over margins for spacing in flex/grid.
* Use `container mx-auto` for centered layouts.
* Use `prose` (Typography plugin) for rich text.
* Use `clsx`/`cn` helper for conditional classes.

---

**Tip:** Build by composing utilities; extract components when repetition shows up.
