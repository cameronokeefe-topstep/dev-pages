## Add a new post

1. Create a file in `_posts/` named `YYYY-MM-DD-title.md`.
2. Add front matter and content, for example:

```markdown
---
layout: post
title: "My New Post"
date: 2026-02-24
---

Write your post content here.
```
3. Click the Commit button

4. Wait a few minutes for the workflow to run and the page will be published automatically.
