## Markdown Cheatsheet

A quick, practical reference for **writing Markdown** (GitHub-flavored where applicable).

---

## 1. Headings

```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

---

## 2. Text Formatting

```md
**Bold**
*Italic*
***Bold & Italic***
~~Strikethrough~~
`Inline code`
```

Result:
- **Bold**
- *Italic*
- ***Bold & Italic***
- ~~Strikethrough~~
- `Inline code`

---

## 3. Paragraphs & Line Breaks

```md
This is a paragraph.

This is another paragraph.
```

Line break (GitHub-flavored): end a line with two spaces.

---

## 4. Lists

### Unordered
```md
- Item
- Item
  - Nested item
```

### Ordered
```md
1. First
2. Second
3. Third
```

### Task Lists (GitHub)
```md
- [x] Done
- [ ] Todo
```

---

## 5. Links

```md
[Link text](https://example.com)
```

### Auto Links
```md
https://example.com
```

---

## 6. Images

```md
![Alt text](image.png)
```

---

## 7. Code Blocks

### Fenced Code Block
```md
```js
console.log('Hello');
```
```

### Inline Code
```md
Use `npm install`
```

---

## 8. Blockquotes

```md
> This is a quote
>> Nested quote
```

---

## 9. Horizontal Rules

```md
---
***
___
```

---

## 10. Tables (GitHub)

```md
| Name | Age | Role |
|------|-----|------|
| Sam  | 30  | Dev  |
| Lee  | 28  | PM   |
```
Result:  
<br>
| Name | Age | Role |
|------|-----|------|
| Sam  | 30  | Dev  |
| Lee  | 28  | PM   |  
  
<br>

Alignment:
```md
| Left | Center | Right |
|:-----|:------:|------:|
```

---

## 11. Escaping Characters

```md
\* \_ \# \`
```

---

## 12. HTML in Markdown

```md
<b>Bold</b>
<br />
```

---

## 13. Emojis (GitHub)

```md
:rocket: :sparkles: :bug:
```

---

## 14. Footnotes (Some Parsers)

```md
Here is a footnote reference[^1].

[^1]: Footnote text
```

---

## 15. Collapsible Sections (GitHub)

```md
<details>
  <summary>Click to expand</summary>

  Hidden content
</details>
```
Result:  
<br>
<details>
  <summary>Click to expand</summary>

  Hidden content
</details>  
<br>

## 17. Common Use Cases

- Notes & wikis

## 18. Markdown Gotchas

- Blank lines matter
- Indentation affects lists & code blocks
- Not all parsers support all features
